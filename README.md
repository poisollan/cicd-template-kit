# 🚀 CI/CD Template Kit

Plug & Play Pipelines for Developers

---

## 💡 What is this?

A production-ready, beginner-friendly CI/CD template kit using:

- GitHub Actions
- Docker
- AWS EC2 deployment

This kit helps you go from **code → deploy** in minutes.

---

## 🎯 Who is this for?

- Developers who want ready CI/CD setup
- DevOps learners
- Freelancers
- Small startups

---

## ⚡ What you get

✔ Docker build & push pipelines  
✔ Node.js CI/CD pipeline  
✔ Python CI/CD pipeline  
✔ EC2 auto-deployment (SSH-based)  
✔ Multi-environment workflow (dev/staging/prod)  
✔ Sample Node.js & Python apps  
✔ Beginner-friendly documentation  

---

## 🔥 Key Features

- Plug & Play setup
- Minimal configuration
- Production-style workflows
- Dockerized deployments
- Real EC2 deployment automation

---

## 🚀 Quick Start

1. Clone this repo
2. Add GitHub Secrets
3. Push code
4. Watch CI/CD run
5. Access your deployed app

---

## 🌐 Example Output

- Node App → `http://your-ec2-ip:3000`
- Python App → `http://your-ec2-ip:5000`
